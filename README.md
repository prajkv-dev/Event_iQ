# SafeFlow AI Navigator

A multi‑page web app to improve entry and exit in high‑profile areas using AI, AR‑style guidance, adaptive QR passes, predictive alerts, and admin analytics. Red/black theme, GitHub Pages‑ready.

## Features
- AI‑powered dynamic routing (simulated)
- AR‑style guidance overlay (camera + canvas arrows)
- Adaptive QR entry passes with congestion‑aware time slots
- Predictive alerts and staggered exit recommendations
- Crowd density heatmap with evacuation mode
- Admin dashboard: gate performance, thresholds, broadcast

## Run locally
1. Clone the repo.
2. Open `index.html` in your browser (no build needed).

## Deploy on GitHub Pages
1. Push the folder to a GitHub repository.
2. In the repo, go to **Settings → Pages**.
3. Select branch `main` and root folder.
4. Your site will be live at `https://<username>.github.io/<repo>/`.

## Notes
- This is a prototype: simulations stand in for real IoT/AI integrations.
- For production, connect to APIs for sensor data, ML predictions, and secure auth.
